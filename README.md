# My-Neuropsych-Chatbot-with-Data-Analysis
Within this project, I focused on creating a chatbot functioning that can give recommendations on physical activity and diet plans for subjects based on the severity of certain markers within this Alzheimer's dataset. This is for training and practices purposes only. Anyone that may want information or know someone that may be affected with memory issues please contact a medical professional immediately.
